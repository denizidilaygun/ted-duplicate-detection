## How to Reproduce

### Setup

```bash
git clone https://github.com/denizidilaygun/ted-duplicate-detection.git
cd ted-duplicate-detection
pip install -r requirements.txt
```

The raw TED dataset (`export_CAN_2023.csv`) must be obtained separately from the [EU Open Data Portal](https://data.europa.eu/data/datasets/ted-csv) due to its size.

### RQ1 — Entity Resolution Pipeline

```bash
cd rq1_entity_resolution/

# 1. Preprocess raw TED data -> 100K stratified sample
python preprocessing.py

# 2. Generate candidate pairs (blocking by CPV + year)
python candidate_pairs.py

# 3. Manually annotate 500 pairs (interactive tool)
python annotation_tool.py

# 4. Synthetic duplicate injection (use v3 for final results)
python synthetic_injection_v3.py

# 5. Feature engineering (string similarity features)
python feature_engineering.py

# 6. Compute SBERT embeddings
python sbert_embeddings.py

# 7. Train and evaluate classifiers (Logistic Regression, Random Forest, XGBoost)
python real_model_training.py

# 8. Evaluate trained model on the 497 manually labelled real pairs (OOD test)
python evaluate_on_real.py

# 9. Generate confusion matrices for the thesis figures
python make_confusion_matrices.py
```

### RQ2 — Generative AI Auditing Pipeline

```bash
cd rq2_genai_audit/

# 1. Prepare 10K subsample for generative model training
python prepare_ted_for_genai.py

# 2. Train CTGAN and TVAE
python train_ctgan.py
python train_tvae.py

# 3. Generate synthetic records
python generate_synthetic.py

# 4. Evaluate fidelity (statistical similarity to real data)
python evaluate_fidelity.py

# 5. Audit memorization using the trained RQ1 classifier
python audit_memorization.py

# 6. Summarize results
python rq2_summary.py
```

### Fairness Analysis

```bash
cd disparate_impact/
python disparate_impact_analysis.py
```

## Synthetic Injection Versions

The synthetic duplicate injection methodology was refined iteratively:

| Version | Perturbation Types | Key Change |
|---------|-------------------|------------|
| v1 | 5 | Initial: case, encoding, abbreviation, word order, word drop |
| v2 | 5 | Added value noise (±15%) and CPV variation (15%) to remove value_ratio artifact |
| v3 | **7** | Added name truncation and punctuation spacing (observed in real TED data) |

The final results reported in the thesis use **v3**.
